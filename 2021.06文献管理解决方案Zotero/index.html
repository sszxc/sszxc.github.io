跨平台文献管理解决方案 Zotero

发表于2021-06-14
|阅读量:

选择 Zotero 的原因

最早的时候使用的文献管理软件是 Mendeley，一开始读的文献不多，

还是存在几个关键问题：同步很慢，非常慢；Mendeley 的 PDF 标注功能不是标准格式，无法在其他平台上浏览；跨平台同步并不方便。

于是这次换成 Zotero，其优势在于：开源轻巧；Chrome/Firefox 插件非常非常方便；简单折腾即可实现跨平台文献管理。

数据迁移过程中
GitHub 找到不少从 mendeley 到 zotero 的导出工具 但好像都需要降级到 1.18
相当复杂 不准备研究了
，就算导出成 PDF 格式也很乱，而且从 1.19 版本开始加密本地数据库了，无法用其他方式读取
好在之前的笔记也不算多，干脆直接丢弃了。

Zotero 的安装

数据同步（文献库的条目、笔记、链接、标签（关键词）） 免费无限制
文件同步（PDF 文档，音频文件，视频文件，图片等） 免费 300M 或使用第三方 WebDAV 网盘

使用坚果云实现文件同步

  • 安装 Zotero，注册坚果云

    参考 用 Zotero+坚果云搞定多设备文献管理 的* 2. 用 Zotero 和坚果云同步至多台电脑*

  • 这一步结束之后可以在多台 PC 之间同步了

  • 实际上这一步换成 Onedrive 也是可以的（如果 iPad 上你常用的 App 支持的话），坚果云的优势在于国内网络下同步速度快很多、支持 WebDAV、不限容量限流量，关键是在iPad上同步快很多

保留同步的文件结构

  • 用上面的这种方式会丢掉文件结构，所以需要安装 Zotfile 插件

    安装步骤参考 在 iPad 上优雅地看文献：Zotero + ZotFile + 坚果云 的* ZotFile 整理 + 坚果云同步*

  • 注意，同步文件的时候要手动操作一下“全选→右键→Manage Attachments→Rename Attachments”，这样才能把附件从“数据存储位置”转移到“Custom Location”

  • subfolder 这里选/%c就可以按分类建立文件夹结构

  • 这一步结束之后坚果云上就会有优雅的文件夹结构了

在 iPad 端优雅地阅读

  • 坚果云网页端开启第三方应用密码，iPad 上 Documents/PDF Expert 添加 WebDAV 服务器

    参考 用 Zotero+坚果云搞定多设备文献管理 的* 3. 用坚果云 webDAV 同步至 iPad*

  • PDF 阅读软件和翻译软件分屏，Apple Pencil 做批注

  • 保存，此时电脑端应该能提示 PDF 文件已更新

关于中文文献的处理方法

有时候会在百度学术、知网上查找一些中文文献，Zotero 的插件并不能很好地捕获相关信息，一个比较快捷的办法是在网页端先“导出 Endnote 引用格式”，然后在 Zotero 中“文件→从剪贴板导入”，最后将 PDF 文件添加到条目中。

另有一个 Zotero 中文插件 Jasminum - 茉莉花 专用于解决中文文献问题，下次一定试试。
